Agile Board - Kanban Task Management App

HTML | CSS | JAVASCRIPT

The Agile Board project showcases my experience building a Kanban Task Management app, completed as a capstone for the JavaScript course. Acting as a junior developer, I enhanced an existing codebase by tackling real-world user stories to fix bugs and expand functionality. My work involved importing utility functions, initializing and persisting data, and using debugging techniques to ensure tasks and boards displayed dynamically and interacted smoothly. In addition, I added new features based on user story requirements, such as managing task details, enabling theme customization, and ensuring data persistence through local storage. This project challenged me to apply critical thinking and problem-solving skills, preparing me for real-world software development.

Shopping List App

HTML | CSS | JAVASCRIPT | FIREBASE

This project demonstrates the integration of Firebase to build a dynamic shopping list app. Users can easily add, edit, and delete items from their shopping list, with all data securely stored and synchronized in real-time using Firebase. The app leverages Firebase's Firestore database to persist items across sessions, ensuring that your list is always up-to-date, even when you switch devices. This project highlights my skills in building responsive, data-driven applications and my experience with real-time databases and backend services like Firebase.
