Agile Board - Kanban Task Management App

HTML | CSS | JAVASCRIPT

Agile Board is a dynamic task management app designed to organize and streamline workflows, built as the capstone project for my JavaScript course. This project highlights my ability to work with existing codebases, solve real-world problems, and implement user-focused features.

Key Features:

  • Dynamic Task and Board Management: Tasks and boards are displayed dynamically, with smooth interactions ensuring an intuitive user experience.
  • Data Persistence: User data is initialized and persisted using local storage, providing a seamless return to previous states.
  • Utility Function Integration: Imported and customized utility functions to enhance app functionality.
  • Debugging and Testing: Applied advanced debugging techniques to resolve bugs and ensure a smooth user experience.
  • Feature Additions: Implemented new features based on user stories, including task detail management and customizable themes.

What I Learned:

Agile Board challenged me to think critically and solve problems in real-world scenarios. I improved my debugging techniques, deepened my knowledge of JavaScript for building interactive web applications, and enhanced my understanding of data management using local storage. This project also provided valuable experience adapting to an existing codebase and extending its functionality with new features. Shopping List App

HTML | CSS | JAVASCRIPT | FIREBASE

This dynamic shopping list app showcases the integration of Firebase to create a real-time, data-driven application for seamless list management.

Key Features:

  • Add, Edit, and Delete Items: Users can easily manage their shopping list with simple and intuitive controls.
  • Real-Time Updates: Firebase's Firestore database ensures changes are synchronized instantly across devices.
  • Data Persistence Across Sessions: The app securely stores and retrieves user data, so your list remains up-to-date, even when switching devices.

What I Learned:

This project enhanced my expertise in working with Firebase, including Firestore for real-time data synchronization and persistence. It also strengthened my ability to create responsive applications that prioritize user experience and security. By integrating backend services, I gained practical knowledge in developing full-stack, real-time solutions that align with modern application standards. ListenUp

HTML | CSS | JAVASCRIPT | TYPESCRIPT | REACT

ListenUp is a fully responsive podcast app designed to showcase my skills in building dynamic and user-focused applications. This project highlights my ability to implement complex features, manage state effectively, and create a seamless user experience with clean and maintainable code.

Key Features:

  • Browsing and Discovery: Users can browse podcasts by genre or search by title and description.
  • Sorting Options: Podcasts can be sorted alphabetically (A-Z or Z-A) or by release date (newest or oldest).
  • Filtering and Favorites: Users can filter podcasts by genre, add episodes to favorites, and manage their favorite episodes easily.
  • Listening Progress Tracking: Includes a built-in audio player that tracks progress, ensuring users can pick up where they left off.
  • Fully Responsive Design: Ensures smooth and optimized usability across all screen sizes.

Technologies Used:

  • React: For building the dynamic and interactive user interface.
  • TypeScript: To ensure type safety and maintainable, bug-resistant code.
  • CSS Modules: For component-specific styling without conflicts.
  • REST API: Fetches real-time podcast data for an up-to-date experience.
  • FontAwesome: Adds modern, professional-looking icons.
  • Netlify: Used for deploying the app for fast and reliable performance.
  • Realfavicongenerator.net: Generated custom favicons for a polished appearance.

What I Learned:

This project allowed me to deepen my understanding of React and TypeScript, focusing on state management, API integration, and user experience design. I also honed my CSS skills by utilizing CSS Modules to maintain organized and reusable styles. Additionally, deploying the app on Netlify taught me how to deliver reliable web applications efficiently. Check out ListenUp to explore how I create dynamic, user-friendly applications that merge functionality with modern design principles.
